### What I've been building

**[SpeechMetrics](https://github.com/22manohar/speechmetrics)** — A vocal quality analyzer that takes an audio file and returns scores across 10 dimensions: clarity, pace, engagement, expressiveness, and more. Built on librosa + Praat for acoustic feature extraction, with a FastAPI layer on top. I built this because every voice coaching tool I found was either a black-box SaaS or gave you a single unhelpful score. This one tells you *exactly* what's off and why.

**[Customer Churn Prediction](https://github.com/22manohar/churn-prediction-project)** — End-to-end pipeline that computes RFM features from retail transaction data and predicts churn with a Random Forest classifier. Covers the full lifecycle: feature engineering, model training, evaluation, and a clean setup for local reproduction.

**[Music & Mental Health Analysis](https://github.com/22manohar/Projects)** — Econometric study on whether music listening habits correlate with self-reported mental health outcomes. Built in Jupyter, includes EDA, statistical modelling, and a full writeup.

**[Core AI Models Journey](https://github.com/22manohar/core-ai-models-journey)** — My scratchpad for implementing ML models from scratch. Linear regression to neural nets. No frameworks, just math and numpy.
